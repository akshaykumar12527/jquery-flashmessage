Custom style

  • Make our own "mitch" type. We just need to have CSS target ".mitch". demo
    $.flashmessage("I'm against picketing, but I don't know how to show it.", { type: 'mitch', timeToFade: 4000 } );
  • Just pass a string of HTML. demo
    var htmlMessage =
    '<em>We\'ll dance like hypnotized chickens.</em><br /><img src="http://www.nndb.com/people/520/000024448/iggy-face.jpg" />';
    $.flashmessage(htmlMessage, { type: 'iggy', timeToFade: 5000 });
